Akari
=================================

.. figure:: /_static/akari-v2.1-heart-200.png
   :align: center


.. centered::
   Providing modern utility to the modern-day Discord support server

Akari provides modern utility to the modern-day Discord support server. Akari takes the most common utilities found in support servers and puts them into one bot, making it easier for you to manage your server.
